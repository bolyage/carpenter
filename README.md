# carpenter
一些踅踅念  
设计模式实现(design patterns) 基于java 实现。

##  在使用某一个模式之前，先思考一下。  

1. 为什么要使用这种模式。  
2. 你用了这个模式之后，带来的利弊分别是什么。  
3. 怎么保证别人能知道你在使用某一个模式？方便后续的开发人员在修改你写的代码的时候能够继续按照你的模式往下写。
